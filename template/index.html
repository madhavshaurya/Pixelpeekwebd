Features

The occupancy counter is a device that can accurately detect and count the number of people in a room. It typically uses sensors to monitor the room's occupancy level and displays the current count on a user interface.

Bad Posture Detection

The system uses sensors to monitor the posture of occupants, and alerts them if they are sitting or standing with bad posture, helping them improve their posture and reduce the risk of injury.

Light Control

The system can automatically adjust the lighting in the room based on the occupancy level, time of day, and natural light levels, improving energy efficiency and reducing eye strain.

Accurate Occupancy Detection

The occupancy counter accurately detects the number of people in a room using sensors, and displays the current occupancy level on a user interface.

Temperature Control

The system can automatically adjust the temperature in the room based on the occupancy level, time of day, and user preferences, ensuring a comfortable working environment.

User Interface

The system includes a user interface that displays occupancy level, temperature, and lighting settings, allowing users to monitor and control the environment.

Easy Installation

The system is easy to install and can be set up quickly in any room, with minimal disruption to occupants.

mechanism

Here are the four steps for the working of the occupancy counter with bad posture detection, light and temperature control system using a webcam as a video source and a Raspberry Pi for processing.

User Interface

The system includes a user-friendly interface that displays the current occupancy level, temperature, and lighting settings. It also displays the bad posture detection parameters and alerts occupants if they need to adjust their posture.

Video Processing

The system uses the OpenCV and Dlib libraries to process the video feed from the webcam. It divides the video feed into four quadrants and uses a combination of face detection and object tracking to accurately detect the number of people in each quadrant and calculate the total occupancy level.

Temperature and Light Control

The system uses a DHT11 sensor to detect the temperature in the room and adjust it as needed to create a comfortable working environment. It also uses four LEDs controlled by GPIO pins to adjust the lighting in the room based on occupancy level and natural light levels.

Bad Posture Detection

The system uses the yawning, nodding, and head tilt parameters to detect bad posture in occupants. It uses machine learning algorithms from the Scikit-learn library to train a model on data collected from the sensors and webcam.
